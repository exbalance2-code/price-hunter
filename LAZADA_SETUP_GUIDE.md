# 🛒 คู่มือการสมัครและตั้งค่า Lazada Open Platform API (สำหรับโปรเจ็ค Chatbot)

คู่มือนี้จะสอนทุกขั้นตอนตั้งแต่เริ่มสมัคร จนถึงการนำค่า Key ต่างๆ มาใส่ในโปรเจ็ค เพื่อให้บอทสามารถค้นหาสินค้าและสร้างลิ้งค์ Affiliate ได้

---

## ขั้นตอนที่ 1: สมัครเป็น Lazada Developer 📝

1. เข้าไปที่เว็บไซต์ [https://open.lazada.com/](https://open.lazada.com/)
2. คลิกปุ่ม **"Login"** (มุมขวาบน)
   - _แนะนำให้ใช้บัญชี Lazada ที่เป็น Affiliate หรือบัญชีที่ตั้งใจจะใช้รับค่าคอมมิชชั่น_
3. หากยังไม่เคยลงทะเบียน Developer ระบบจะให้กรอกข้อมูลผู้พัฒนา ให้กรอกตามความจริง

---

## ขั้นตอนที่ 2: สร้าง Application (App) 📱

เพื่อให้ได้ `App Key` และ `App Secret` เราต้องสร้าง App ขึ้นมาใหม่ 1 ตัว

1. คลิกเมนู **"Console"** (มุมขวาบนหลัง Login)
2. เลือกเมนู **"App Management"** -> คลิก **"Create App"**
3. เลือกประเภทของ App:
   - กรณีเป็น Affiliate ทั่วไป: มักจะอยู่ในหมวด **"Tools"** หรือ **"Affiliate"** (ลองหาหมวดที่เขียนว่า _Affiliate_ หรือ _Promote_)
   - หากไม่เจอ ให้เลือก **"Utility"** หรือประเภทที่ใกล้เคียง
4. กรอกรายละเอียด:
   - **App Name:** ตั้งชื่ออะไรก็ได้ (เช่น `PriceHunterBot`)
   - **Description:** ใส่คำอธิบายสั้นๆ (เช่น `LINE Chatbot for product search`)
   - **Callback URL:** `https://www.google.com` (ใส่หลอกไว้ก่อนได้ เพราะเราจะ Copy Token ด้วยมือ)
5. กด **Submit**

---

## ขั้นตอนที่ 3: ขอสิทธิ์ (Apply Permission) 🔓

_ขั้นตอนนี้สำคัญมาก เพื่อให้ App เรามีสิทธิ์ดึงข้อมูลสินค้า_

1. ในหน้า App ของเรา ไปที่เมนู **"App Overview"**
2. เลื่อนลงมาดูส่วน **"Permission Group"** หรือ **"API Group"**
3. กด **"Apply"** ในหมวดที่ชื่อว่า:
   - **`Lazada Affiliate`** (สำคัญที่สุด!)
   - หรือ **`Marketing`**
   - หรือ **`Product`** (สำหรับดูข้อมูลสินค้า)
4. รอการอนุมัติ (ปกติถ้าเป็น User ทั่วไปอาจจะอนุมัติทันที หรือรอ 1-2 วันทำการ)
   - _สังเกตสถานะ App ต้องเป็น **"Online"** หรือ **"Normal"** ถึงจะใช้งานได้จริง_

## ขั้นตอนที่ 5: สร้าง Access Token (หัวใจสำคัญ) ❤️

ขั้นตอนนี้คือการสร้าง "กุญแจ" สำหรับเข้าถึงข้อมูลจริง

1. ไปที่เมนู **"Tools"** (แถบเมนูด้านบน) -> เลือก **"API Playground"** (หรือ API Test Tool)
2. **ตั้งค่าการทดสอบ:**
   - **App:** เลือกชื่อ App ที่เราเพิ่งสร้าง
   - **API Category:** เลือกหมวด `Marketing` หรือ `Affiliate`
   - **API Name:** เลือก `/marketing/product/feed` (หรืออันไหนก็ได้)
3. **Generate Token:**
   - มองหาปุ่มที่เขียนว่า **"Get Access Token"** หรือ **"Authorise"**
   - ระบบจะเด้งหน้าต่างให้เรา Login บัญชี Lazada (บัญชีผู้ซื้อ/Affiliate)
   - พอกดอนุญาต ระบบจะแสดงค่า:
     - **`access_token`:** (รหัสยาวๆ นี่คือสิ่งที่เราต้องการ!)
     - **`refresh_token`:** (เก็บไว้ใช้ต่ออายุเมื่อ access_token หมดอายุ)
4. **Copy `access_token` เก็บไว้**

## ขั้นตอนที่ 6: นำค่ามาใส่ในโปรเจ็ค Price Hunter 🏗️

### วิธีที่ 1: ใส่ใน Render (Cloud)

1. ไปที่ Dashboard ของ Render -> เลือกโปรเจ็ค `price-hunter-bot`
2. เมนู **Environment**
3. แก้ไข/เพิ่มค่าตามนี้:
   - `LAZADA_APP_KEY`: (ค่าจากขั้นตอนที่ 4)
   - `LAZADA_APP_SECRET`: (ค่าจากขั้นตอนที่ 4)
   - `LAZADA_ACCESS_TOKEN`: (ค่าจากขั้นตอนที่ 5)
4. กด **Save Changes** (ระบบจะ Deploy ใหม่ให้อัตโนมัติ)

### วิธีที่ 2: ใส่ในไฟล์ทดสอบ (เพื่อความชัวร์ก่อน)

1. เปิดไฟล์ `scripts/verify-lazada-token.ts` ใน VS Code
2. แก้ไขค่าตัวแปรด้านบนสุดให้ตรงกับของใหม่:
   ```typescript
   const APP_KEY = "...ค่าใหม่...";
   const APP_SECRET = "...ค่าใหม่...";
   const ACCESS_TOKEN = "...ค่าใหม่...";
   ```
3. รันคำสั่งทดสอบ: `npx tsx scripts/verify-lazada-token.ts`
4. ถ้าขึ้น **✅ SUCCESS** แปลว่าทุกอย่างพร้อมใช้งานแล้ว!

---

### ⚠️ ข้อควรระวัง

- **Access Token มีวันหมดอายุ:** (ปกติ 7 หรือ 30 วัน) ถ้าบอทใช้งานไม่ได้ ให้มาทำขั้นตอนที่ 5 ใหม่เพื่อเอา Token ใหม่ไปใส่
- **App Key ต้องตรงกับ Token:** ห้ามเอา Token ของ App A มาใช้กับ App Key B จะ Error ทันที
